# . is scaffolded with [Initex](https://www.npmjs.com/package/initex).

## How to run?

1. Install dependencies:

   ```bash
   npm install
   ```
2. Setup the database:

   ```bash
   npm run db:up
   ```

3. Run the project:

   ```bash
   npm run dev
   ```

4. Open http://localhost:8000 in your browser to view the project.
